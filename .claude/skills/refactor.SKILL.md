# Refactor Skill

## When to Use
- Type unification tasks
- Adapter removal
- Code modernization

## Steps
1. Identify scope and create plan
2. Run TypeScript check before changes
3. Make incremental changes
4. Run tests after each change
5. Verify no regressions

## Safety
- Never modify tungsten-tiff.ts
- Always check TypeScript errors
- Commit after each working state
